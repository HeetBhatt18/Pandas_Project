# 📊 Pandas Data Analysis Project (Titanic Dataset)

## 📌 Overview

This project demonstrates data analysis using Python Pandas on the
Titanic dataset. It covers data exploration, cleaning, selection, and
statistical analysis.

## 🚀 Features

-   Data exploration
-   Data cleaning
-   Data selection
-   Statistical analysis

## 🗂️ Project Structure

Pandas_Project.ipynb \# Main notebook Titanic.csv \# Dataset (if used)
README.md \# Documentation

## 📥 Dataset

Titanic dataset including: - Age - Fare - Survival - Passenger class
